En el siguiente caso nos muestra la salida con error y el error en cuestión:
``` yaml
--- #ignore_errors.yml
- name: Comprobar fichero
  hosts: python
  become: False
  tasks:
    - name: Visualizar variable
      command: cat /etc/hosts2
      register: salida
      ignore_errors: True #Esto hará que podamos continuar con la siguiente tarea si esta tarea nos ha dado error

    - name: Visualizar variable
      debug: var=salida.stdout_lines

    - name: Visualizar errores
      debug: var=salida.stderr
```
Se puede usar la condición `when: salida.rc == 0` para que solo se ejecute si el fichero **EXISTE**.
También podemos poner que solo se ejecute si el fichero **NO EXISTE** `when: salida.rc != 0`.
Un ejemplo práctico es el siguiente:
``` bash
--- #ignore_errors.yml
- name: Comprobar fichero
  hosts: python
  # become: True
  tasks:
    - name: Variable fichero
      command: cat /etc/hosts2
      register: salida
      ignore_errors: True
    - name: Visualizar variable (si el fichero existe)
      debug:
        var: salida.stdout_lines
      when: salida.rc == 0
    - name: Visualizar errores (si el fichero no existe)
      debug:
        var: salida.stderr
      when: salida.rc != 0
```
