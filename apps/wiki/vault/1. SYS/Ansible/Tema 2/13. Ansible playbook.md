Ejecutar playbook:
``` Bash
ansible-playbook -i inventarios/ playbooks/playbook.yaml
```
Comprobar sintaxis:
``` Bash
ansible-playbook -i inventarios/ playbooks/playbook.yaml --syntax-check
```
Listar hosts:
``` Bash
ansible-playbook -i inventarios/ playbooks/playbook.yaml --list-hosts
```
Listar tareas:
``` Bash
ansible-playbook -i inventarios/ playbooks/playbook.yaml --list-tasks
```
Realizar un debug para confirmar paso a paso lo que va a realizar el fichero **.yaml**:
``` Bash
ansible-playbook -i inventarios/ playbooks/playbook.yaml --step
```
Para empezar desde una tarea específica usamos `--start-at-task="[tarea]"`:
``` Bash
ansible-playbook -i inventarios/ playbooks/playbook.yaml --start-at-task="Ping a servidores Linux"
```
Para realizar varias tareas en paralelo usamos `--forks=[nº] o -f [nº]`:
``` Bash
ansible-playbook -i inventarios/ playbooks/playbook.yaml --forks=1
```
Para ver más información (usar **verbose**) usamos `-v` o `-vvv`, para ver comandos que se usan para establecer la conexión:
``` Bash
ansible-playbook -i inventarios/ playbooks/playbook.yaml -v
```
