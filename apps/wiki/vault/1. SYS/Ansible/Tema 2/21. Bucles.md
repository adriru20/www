Para listas o directorios que vayamos a crear usaremos **with_items** y pondremos {`{ item }}` como variable que va recorriendo la lista **with_items**.

Ej. de instalación de varios paquetes:
``` yaml
--- #bucles.yml
- name: Pruebas con bucules
  hosts: testing
  become: true
  tasks:
    - name: Instalar software por defecto
      apt: name={{ item }} state=latest
      with_items:
        - tree
        - figlet
        - sl
```
Ej. de uso de variables con roles, creación de varios usuarios:
```
roles/usuarios/
├── defaults
│   └── main.yml
└── tasks
    └── main.yml
```
``` yaml
--- #bucles_crear-usuarios.yml ++roles/defaults/main.yml & ++roles/tasks/main.yml
- name: Pruebas con bucles
  hosts: python
  become: true
  roles:
    - usuarios
```
``` yaml
--- #roles/defaults/main.yml
# lista_usuarios: ["test1","test2"]
dic_usuarios:
  - {nombre: "usuario1", grupo: "user"}
  - {nombre: "usuario2", grupo: "user"}
```
``` yaml
--- #roles/tasks/main.yml
- name: Crear usuariuos
  user: name={{ item.nombre }} groups={{ item.grupo }}
  with_items: "{{ dic_usuarios }}"
```
Ej. de uso en un solo fichero, eliminación de usuarios:
``` yaml
--- #bucles_crear-usuarios.yml
- name: Pruebas con bucles
  hosts: python
  become: true
  tasks:
    - name: Borrar usuarios de prueba
      user: name={{ item }} state=absent remove=yes
      with_items:
        - test1
        - test2
        - usuario1
        - usuario2
```
