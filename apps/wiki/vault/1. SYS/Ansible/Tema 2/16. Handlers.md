# Info:
- Handlers (parecido a una función) es una tarea que solo se ejecuta cuando otra tarea la llama.
- Siempre se ejecutan al final de cada playbook.
## Ejemplo:
``` yaml
tasks:
    - name: configurar sshd_config
      copy: src=sshd_config dest=/etc/sshd_config
  notify: restart_sshd
handlers:
    - name: restart_sshd
  service: name=sshd state=restarted
```
