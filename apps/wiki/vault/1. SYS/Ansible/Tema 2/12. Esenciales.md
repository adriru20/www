A la hora de seleccionar distintos hosts desde el campo <span style="color:rgb(0, 176, 240)">hosts:</span> usaremos <span style="color:rgb(255, 192, 0)">:</span> para añadir los siguientes atributos, al usar <span style="color:rgb(0, 176, 80)">&</span> (se hará en todos los **grupos/servidores** que se añadan) o <span style="color:rgb(255, 0, 0)">!</span> (se hará en el primero, pero no en el segundo **grupo/servidor**), ambos tienen que iniciar con.
# Ejemplos:
``` yaml
hosts: debian
hosts: debian:&local
```
# Parámetros en playbooks:
``` yaml
hosts: [servidores]
remote_user: [user]
become: [True | False]
become_user: [user]
become_method: [su | sudo]
check_mode: [True | False] #True para comprobar en que servidores se realizará la acción y False para realizar la acción directamente.
```
