# Sintaxis:
Para establecer el propietario del fichero: `owner: root`
Para establecer el grupo del fichero: `group: root`
Para establecer los permisos del fichero: `mode: 0644`
## Ejemplos con variables y atributos:
``` yaml
- name: Crear fichero usando variables
  hosts: localhost
  connection: local
  vars:
    - miip: '1.2.3.4'
    - lista_horizonal: ["apache2","mariadb","php5"]
    - lista_vertical:
      - apache2
      - mariadb
      - php5
  tasks:
    - name: Crear fichero hosts
      template: src=../vars/hosts.j2 dest=/temp/hosts owner=root group=root mode=0644
    - name:
      template:
        src:   ../vars/hosts.j2
        dest:  /temp/hosts
owner: root
group: root
mode:  0644
```
## Ejemplo de Diccionario:
``` yaml
- name: Crear fichero usando variables
  hosts: localhost
  connection: local
  vars:
    - miip: '1.2.3.4'
    - midic_horizontal: {web: "apache2", bd: "mariadb", script: "php5"}
- midic_vertical:
- web: apache2
- bd: mariadb
- script: php5
```
## Ejemplos de Textos:
``` yaml
- name: Crear fichero usando variables
  hosts: localhost
  connection: local
  vars:
    - miip: '1.2.3.4'
- textolineas: |
primera linea
segunda linea
tercera linea
- textolinea: >
inicio linea
continuación linea
final linea
Uso del debug:
- name: Crear fichero usando variables
  hosts: localhost
  connection: local
  vars:
    - miip: '1.2.3.4'
    - lista_horizonal: ["apache2","mariadb","php5"]
    - lista_vertical:
      - apache2
      - mariadb
      - php5
  tasks:
- debug: var=miip
    - debug: var=lista_horizonal
    - debug: var=lista_vertical
    - debug: var=midic_horizontal
    - debug: var=midic_horizontal.web #Para solo sacar un valor del diccionario.
    - debug: var=midic_vertical
    - debug: var=textolineas
    - debug: var=textolinea
```
