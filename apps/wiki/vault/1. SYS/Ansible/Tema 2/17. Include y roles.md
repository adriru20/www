# Include:
``` yaml
--- #include.yml
- name: Primer play
  hosts: python
  become: true
  tasks:
    - name: Actualizar la cache de APT
      apt:
        update_cache: yes
    - name: Instalar vim
      apt:
        name: vim
        state: latest
      - include_tasks: /etc/ansible/playbooks/instalar_apache2.yml
--- #instalar_apache2.yml
- name: Instalar apache2
  apt: name=apache2 state=latest
- name: Iniciar servicio
  service: name=apache2 state=started
- name: Habilitar servicio
  service: name=apache2 enabled=true
```
# Roles:
- Existe un repositorio llamado "Ansible Galaxy" donde aparecen roles ya creados por la comunidad.
Un ejemplo de "tree" de roles sería el siguiente:
**roles**/
	nombre/
		**files**/<ficheros de configuración o para copiar en los clientes>
		**templates**/main.yml
		**tasks**/main.yml
		**handles**/main.yml
		**vars**/main.yml
		**defaults**/main.yml
		**meta**/main.yml
## En el fichero, normal:
``` yaml
roles:
 - rol1
 - rol2
```
## En el fichero, con variables:
``` yaml
roles:
 - {role: rol1, clave: valor}
roles:
 - {role: rol1, usuario: apache2}
```
## Fichero de ej.
``` yaml
--- #roles.yml
- name: Instalar y configurar apache2
  hosts: python
  become: true
```
