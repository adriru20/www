## Curso
- Enlace: [Curso de Ansible: Automatización de principiante a experto.](https://www.udemy.com/course/ansible-automatizacion-de-principiante-a-experto/)
- Coste: 89,99€ -> 12,99€
- Clases: 70
- Horas: 10h
- Idioma: Español.
## Ventajas
- No requiere un agente.
- Sintaxis simple y fácil de aprender.
- Seguro y fácil de mantener (SSH).
- Rendimiento.
- No requiere saber programar (+Saber Python para crear módulos).
## Desventajas
- No es potente como administrador de configuraciones (Usar Git para control de versiones).
- Requiere en grandes entornos configuraciones avanzadas.
- Bugs o errores con soporte se suele tener que esperar a que lo arreglen en la siguiente versión disponible.
