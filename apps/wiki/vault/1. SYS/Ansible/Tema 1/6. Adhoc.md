# Atributos del prompt
Para limitar equipos usamos `-l` | `--limit`:
``` Bash
ansible -i inventarios/prod all -l docker,ionos -a "cat /etc/debian_version"
```
Para establecer el usuario usamos `-u` | `--user`:
``` Bash
ansible -i inventarios/prod all -u ansible -m ping
```
Para usar sudo o su usamos `-b` | `--become`:
``` Bash
ansible -i inventarios/prod all -b -a id
```
Para establecer un límite de equipos al mismo tiempo distinto a **5** usamos `-f`:
``` Bash
ansible -i inventarios/prod all -f 1 -m ping
```
Para sacar un listado de todos los host establecidos en el inventario usamos `--list-hosts`:
``` Bash
ansible -i inventarios/prod all --list-hosts
```
Para comprobar acciones antes de realizarlas usamos `-C`:
``` Bash
ansible all -i inventarios/prod -l ionos -m copy -a "src=/mnt/copiame dest=/root/copiame2" -C
```
Para ver más detalles usamos el parámetro `-v` y para ver todo lo que sucede usaremos `-vvvv`:
``` Bash
ansible all -i inventarios/prod -m ping -vvvv
```
# Módulos
El módulo **setup** sirve para ver todos los datos de los servidores:
``` Bash
ansible -i inventarios/prod all -l docker -a "cat /etc/debian_version" -m setup | less
```
El módulo **copy** sirve para copiar ficheros con los atributos de salida (Ansible) y destino (Servidor) `"src=/ dest=/"`:
``` Bash
ansible all -i inventarios/prod -l ionos -m copy -a "src=/mnt/copiame dest=/root/copiame"
```
El módulo **apt/yum** sirve para *instalar*, *actualizar*, *eliminar* y *comprobar* paquetes con los atributos de nombre y el estado del paquete (Para comprobar si está o no instalado) `"name=paquete state=[present | absent]"`:
``` Bash
ansible -i inventarios/dev debian -l python -u root -b -m apt -a "name=tree state=present"
```
