# Valores y configuraciones:
- Valores por defecto globales
- Conf. de escalado de permisos
- Opciones de OpenSSH
- Opciones SELinux
- Conf. Ansible Galaxy
# Orden de búsqueda:
- Variable de entorno: **ANSIBLE_CONFIG**  
- Ficheros de configuración en el directorio actual: **ansible.cfg** y **.ansible.cfg**
- Fichero por defecto: **/etc/ansible/ansible.cfg**
# Configuraciones fichero ansible.cfg:
| CONFIG                          | DETALLES                                                   |
| ------------------------------- | ---------------------------------------------------------- |
| roles_path = /etc/ansible/roles | Ruta para fichero de roles.                                |
| host_key_checking = False       | Preguntar si queremos confiar en una nueva clave SSH o no. |
| remote_user = root              | Usuario por defecto al que queremos conectarnos.           |
| log_path = /var/log/ansible.log | Activar o desactivar los logs.                             |
| max_diff_size = 1048576         | Establecer el uso de RAM máxima del servidor.              |

| SECCIONES              | FINALIDAD                                          |
| ---------------------- | -------------------------------------------------- |
| [privilege_escalation] | Configurar el atributo --become o -b.              |
| [connection]           | Configurar los parámetros de SSH.                  |
| [selinux]              | Configurar, si lo usamos, SELinux.                 |
| [colors]               | Configurar la paleta de colores en los resultados. |
