# Configuración de servidor ansible y equipos:
- Lista de elementos a administrar: /etc/ansible/hosts
- Conexión por defecto: SSH
- Para conexión local: ansible_connection=local
- Para utilizar un usuario en concreto: ansible_user=usuario
- Para la conexión remota por SSH:
	- Host ansible: 
		``` bash
		ssh-keygen; cat ~/.ssh/id_rsa.pub #Copiamos el resultado en el fichero cliente.
		```
	- Host cliente:
		``` Bash
		sudo vi .ssh/authorized_keys #Y pegamos la clave.
		```
- Comprobar conexión (`-m = module`):
	``` Bash
	ansible [server | grupos] -m ping`
	```
- Lanzar comandos:
	``` Bash
	ansible [server | grupos] -a hostnme
	```
- Para lanzar comandos en la lista de `/etc/ansible/host`:
	``` Bash
	ansible all -a hostname
	```
- Para usar sudo pondremos al final `--become`:
	``` Bash
	ansible -m user -a "name=user state=present" --become
	```
- Para ver los usuarios que usa cada host usaremos:
	``` Bash
	ansible all -a "id"
	```
