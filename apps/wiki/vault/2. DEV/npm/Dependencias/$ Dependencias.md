Todas las dependencias se almacenan en la carpeta `node_modules`.

1. Instala jQuery:
``` Bash
npm install jquery
```

2. Instala la dependencia de Bootstrap, la ruta sería: "/bootstrap/dist/...":
``` Bash
npm install -save bootstrap
```

3. Crea un servidor en local para poder ver el proyecto:
``` Bash
npm install http-server --save--dev
```

4. Instala las dependencias de http-serve:
``` Bash
npm install http-server --save--dev -g
```

5. Para instalar algo como http-server pero sin ser local.
``` Bash
npm i -g cloudflared
```

6. Para buscar dependencias de lo que pongamos:
``` Bash
npm search angular
```
